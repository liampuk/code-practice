# no14

Given an array of ints, return true if it contains no 1's or it contains no 4's.

```
no14([1, 2, 3]) → true
no14([1, 2, 3, 4]) → false
no14([2, 3, 4]) → true
```

**Solution:**

```java
public boolean no14(int[] nums) {
  boolean containsOne = false;
  boolean containsFour = false;
  for(int i=0; i<nums.length; i++){
    if(nums[i] == 1){
      containsOne = true;
      if(containsFour){
        return false;
      }
    }if(nums[i] == 4){
      containsFour = true;
      if(containsOne){
        return false;
      }
    }
  }
  return true;
}
```
