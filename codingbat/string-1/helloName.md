# helloName

Given a string name, e.g. "Bob", return a greeting of the form "Hello Bob!".

```
helloName("Bob") → "Hello Bob!"
helloName("Alice") → "Hello Alice!"
helloName("X") → "Hello X!"
```

**Solution:**

```java
public String helloName(String name) {
  return "Hello " + name +"!";
}
```
